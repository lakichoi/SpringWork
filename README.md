# springWork

## A. 개요
Spring FrameWork를 활용한 게시판 CRUD와 AJAX/JSON을 활용한 RestAPI 방식의 댓글시스템 구현.

## B. 구현목록
- Board
1.게시판 목록   
2.게시글 읽기    
3.게시글 쓰기     
4.게시글 수정   
5.게시글 삭제   

- Reply
6.댓글 목록   
7.댓글 읽기   
8.댓글 쓰기   
9.댓글 수정   
10.댓글 삭제
   
   
초기화면 : 로그인 / 회원가입 페이지로 이동
---   
<br>   

회원가입 : 회원등록 / 로그인페이지로 이동
---   
<br>   

회원목록 : 로그인시 나타나는 회원목록 / 헤더부분에 로그아웃 버튼 활성화
---   
<br>   

회원수정: 회원정보 수정창 / 회원삭제 버튼
---
<br>
